# 🔍 System Audit Tool (TSR-001)
> **Baseline diagnostic utility for system integrity and resource auditing.**

### 📋 Overview
Esta herramienta es un activo técnico de **Tapatío Systems Research** diseñado para la recolección automatizada de metadatos de entorno. Facilita la auditoría inicial de seguridad necesaria para procesos de hardening de infraestructura y observabilidad.

### 🛠️ Functionality
* **Environment Mapping:** Identificación de arquitectura, OS y privilegios de ejecución.
* **Security Surface Audit:** Verificación de integridad de archivos críticos.
* **Operational Logging:** Diagnósticos base para análisis de resiliencia.

### 🚀 Usage
Requiere Python 3.x.

python3 audit.py
